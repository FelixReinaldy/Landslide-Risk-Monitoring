# Landslide Risk Simulation System

A Windows-based simulation tool that models multi-sensor fusion and landslide risk dynamics across time, frequency, S-domain, and Z-domain.

## Features
- Real-time virtual sensor simulation  
- FFT-based frequency analysis  
- First-order dynamic risk model  
- Multi-sensor fusion visualization  
- Interactive WinForms GUI  

## Tech Stack
- C# (.NET Framework)
- WinForms
- Visual Studio
